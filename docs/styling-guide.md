# 服装道具搭配知识库设计 (Styling & Props Guide)

> 本文档定义服装搭配建议和道具推荐的数据结构、匹配逻辑、知识库体系及商业化联动方案。

---

## 一、设计目标

在推荐姿势的同时，告诉用户：**穿什么颜色显白、什么款式遮肉、拿什么道具不尴尬。** 服装-道具-姿势-参数四项联动，形成完整的前期准备建议。

---

## 二、服装搭配知识库

### 2.1 服装色彩搭配规则

#### 场景主色调 → 服装颜色推荐

| 场景主色调 | 推荐颜色（出彩） | 推荐颜色（安全） | 避免 |
|------------|------------------|------------------|------|
| 蓝色（海/天空） | 白色、黄色、橙色 | 浅蓝、米色 | 深蓝（融入背景）、黑 |
| 绿色（草地/森林） | 白色、红色、浅粉 | 米色、卡其 | 深绿（融入）、荧光色 |
| 灰色（工业风/建筑） | 红色、蓝色、黄 | 白色、黑、灰 | — |
| 米色/土色（沙漠/古城） | 白色、红色、宝蓝 | 棕色、焦糖 | 土黄（融入） |
| 粉色（花田/樱花） | 白色、浅蓝、浅紫 | 米色 | 粉色（融入）、大红 |
| 白色（雪/白墙） | 任何鲜艳色 | 任何色 | —（白色是万能背景） |
| 黑色/暗色（夜景） | 金色、银色、白 | 红、亮粉 | 黑（融入）、深蓝 |
| 霓虹灯混合 | 白色（最佳）、黑色（剪影） | 单色纯色 | 花纹/条纹 |

#### 肤色 → 服装颜色建议

| 肤色类型 | 推荐色系 | 避免色系 |
|----------|----------|----------|
| 偏白/冷白皮 | 任何色都友好，尤其蓝色系、粉色系 | — |
| 偏黄/暖黄皮 | 蓝色系、紫色系、酒红、墨绿 | 荧光黄、橙色、土黄 |
| 偏黑/小麦皮 | 白色、亮色（红/橙/黄）、金属色 | 暗灰、暗棕 |
| 偏粉/泛红 | 冷色调（蓝/绿/紫）、白色 | 红色、橙色 |

### 2.2 服装风格匹配规则

| 场景氛围 | 推荐服装风格 | 示例单品 |
|----------|-------------|----------|
| 自然/田园 | 清新/日系/森女 | 碎花裙、草帽、帆布鞋 |
| 都市/摩登 | 简约/高级感/性冷淡 | 西装外套、阔腿裤、纯色衬衫 |
| 工业/酷感 | 暗黑/街头/酷飒 | 皮衣、马丁靴、深色牛仔裤 |
| 优雅/古典 | 法式/复古/港风 | 丝绒裙、珍珠配饰、中跟鞋 |
| 度假/休闲 | 慵懒/波西米亚/度假 | 长裙、吊带、凉鞋 |
| 运动/活力 | 运动休闲/athleisure | 卫衣、运动鞋、棒球帽 |

### 2.3 场景-服装适配规则

| 场景 | 实用限制 | 推荐面料 |
|------|----------|----------|
| 海边 | 避免短裙（风大）、细高跟（沙子） | 飘逸长裙、阔腿裤、平底凉鞋 |
| 草地 | 避免细高跟（陷进土）、白色长裙（草汁） | 厚底鞋、牛仔裤、中长裙 |
| 山顶 | 避免高跟鞋、短裙（冷/危险） | 运动鞋、长裤、防风外套 |
| 咖啡厅 | 无特殊限制 | 日常穿搭即可 |
| 雨天 | 避免长裙（拖地）、麂皮（怕水） | 短靴、防水外套、透明伞 |
| 雪地 | 避免深色厚重羽绒服（臃肿） | 亮色外套、毛线帽、围巾 |

## 三、道具推荐知识库

### 3.1 道具分类

```
道具分类
├── 随身常备（不需要额外购买）
│   ├── 手机
│   ├── 耳机
│   ├── 墨镜
│   ├── 包包
│   └── 饮品杯
├── 季节场景
│   ├── 春天: 花束、气球、泡泡机
│   ├── 夏天: 草帽、墨镜、纱巾、冰淇淋
│   ├── 秋天: 落叶、热饮杯、围巾、书本
│   └── 冬天: 围巾、手套、热可可、仙女棒
├── 场景专用
│   ├── 海边: 贝壳、冲浪板、遮阳伞
│   ├── 街拍: 咖啡杯、自行车、报纸、滑板
│   ├── 咖啡厅: 书本、笔电、蛋糕
│   ├── 公园: 花束、野餐篮、气球
│   └── 夜景: 仙女棒、灯笼、霓虹灯、透明伞
└── 特殊道具
    ├── 乐器（吉他/尤克里里）
    ├── 宠物
    ├── 相机（做道具）
    └── 车（汽车/单车做背景）
```

    "discount_code": "POSECRAFT10"

### 5.2 道具电商导流

- 推荐道具卡片底部显示购买链接（跳转淘宝/京东/拼多多）
- 分佣模式（CPS）：用户通过 APP 跳转购买后抽佣 3-10%
- 道具自营商城（Phase 3）：自有供应链，更高毛利

## 六、知识库数据来源

| 来源 | 方式 | 质量 |
|------|------|------|
| 时尚博主/造型师合作 | 付费邀请撰写搭配方案 | ⭐⭐⭐⭐⭐ |

| 色彩理论规则化 | 将色彩搭配理论（互补色/邻近色/冷暖对比）转译成规则 | ⭐⭐⭐⭐ |
| LLM 生成 + 人工审核 | Qwen-VL 识别照片中的服装 → DeepSeek 生成搭配建议 → 人工打磨 | ⭐⭐⭐ |
| 用户身材反馈 | 用户对推荐搭配的满意度 → 优化体型-服装匹配算法 | ⭐⭐⭐ |
