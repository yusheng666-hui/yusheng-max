# 相机参数知识库设计 (Camera Parameter Knowledge Base)

> 本文档定义相机参数的分类体系、推荐逻辑、姿势-参数联动规则以及各手机品牌的参数能力映射。

---

## 一、设计目标

让 APP 不仅说"站这、这么摆"，还能说"用这个参数拍"。**同一个姿势在不同光线条件下，需要完全不同的参数组合。**

## 二、参数分类体系

### 2.1 两级粒度：小白模式 vs 进阶模式

```
用户水平判断依据：
├── 注册时自选（小白 / 爱好者 / 专业）
├── 使用中行为推断：
│   ├── 是否使用过专业模式拍照？
│   ├── 是否手动调整过 EV？
│   └── 历史照片中 EXIF 数据是否有手动设置痕迹？
└── 用户可在拍照界面随时切换模式
```

| 参数 | 小白模式 | 进阶模式 |
|------|----------|----------|
| 拍摄模式 | "打开人像模式" | — |
| HDR | "打开 HDR" 或 "关闭 HDR" | — |
| 闪光灯 | "不用闪光灯""需要补光" | 补光闪光灯 + 具体强度 |
| 夜景模式 | "用夜景模式，保持手机稳定" | 夜景模式 + 堆栈张数建议 |
| ISO | — | ISO 100-6400 具体数值 |
| 快门速度 | — | 1/8000s — 30s 具体数值 |
| EV 曝光补偿 | — | -3EV — +3EV |
| 白平衡 | — | 色温 K 值 或 预设名 |
| 对焦点 | — | "对焦在眼睛""对焦在人脸" |
| 测光模式 | — | 矩阵 / 中央重点 / 点测光 |
| 色彩配置 | — | 鲜艳 / 自然 / 人像 / 黑白 |
| RAW | — | 是否建议拍 RAW |

### 2.2 参数决策树

```
场景分析
    │
    ├── 光线强度 → 决定 ISO 基准 + 快门速度下限
    │   ├── 强光 (EV>13): ISO 100, 快门≥1/500s
    │   ├── 正常 (EV 10-13): ISO 100-400, 快门≥1/125s
    │   ├── 弱光 (EV 7-10): ISO 400-1600, 快门≥1/60s
    │   └── 暗光 (EV<7): ISO 800-6400, 需要三脚架/夜景模式
    │
    ├── 光线方向 → 决定测光模式 + EV 补偿 + HDR
    │   ├── 顺光: 矩阵测光, EV 0
    │   ├── 侧光: 中央重点, EV -0.3（保留高光）
    │   ├── 逆光: 点测光(人脸), EV +0.7 或拍剪影 EV -2
    │   └── 顶光: 人脸点测光, EV +0.3, 建议找阴影
    │
    ├── 主体运动 → 决定快门速度 + 对焦模式 + 连拍
    │   ├── 静止: 快门安全速度即可, AF-S
    │   ├── 慢速运动(走路): 快门≥1/250s, AF-C
    │   ├── 快速运动(奔跑/跳跃): 快门≥1/1000s, AF-C + 连拍
    │   └── 长曝光创意(水流/车轨): 快门 1-30s, 需三脚架, 手动对焦
    │
    ├── 色彩温度 → 决定白平衡
    │   ├── 日光 5500K → 自动或 5200-5500K
    │   ├── 阴天 6500K → 阴天预设或 6200-6500K
    │   ├── 钨丝灯 3200K → 钨丝灯预设或保持 3200K（保留暖色氛围）
    │   ├── 夜景混合光源 → 自动白平衡
    │   └── 日出/日落 → 日光预设，保持金色暖调
    │
    ├── 场景动态范围 → 决定 HDR 策略
    │   ├── 天空+地面高反差 → HDR 开
    │   ├── 均匀光照 → HDR 不需要
    │   ├── 逆光人像 → HDR 开 + EV +0.3
    │   └── 如需 RAW → 关 HDR，后期合成
    │
    └── 用户水平 + 设备限制 → 最终输出粒度
```

## 四、姿势-参数联动规则 ⭐

### 4.1 核心原则

**同一个场景下，不同姿势需要不同参数。** 这是 PoseCraft 区别于任何相机 APP 的独有功能。

### 4.2 联动维度

| 姿势特征 | 影响参数 | 示例 |
|----------|----------|------|
| **动态/静态** | 快门速度、对焦模式、连拍 | 跳起→1/1000s + AF-C + 连拍；站着不动→1/125s + AF-S |
| **面向/背对光源** | 测光模式、EV、HDR | 正对太阳→点测人脸 +0.7EV；背对太阳→矩阵测光 0EV |
| **全身/半身/特写** | 光圈、焦距、景别 | 半身→人像模式(大光圈)；全身→缩小光圈保证清晰 |
| **手部动作** | 快门速度 | 手拿道具静止→1/60s 安全快门；抛东西→1/1000s 冻结动作 |
| **表情** | 对焦点 | 正脸→眼睛对焦；侧脸→更近的眼睛；闭眼→人脸框对焦 |
| **头发/裙摆飘动** | 快门速度 + 连拍 | 需要自然飘动→1/500s + 连拍 3-5 张 |

## 六、手机品牌专业模式能力映射

不同 Android 手机的"专业模式"能力差异很大，参数推荐需要感知设备能力。

| 参数 | 华为 | 小米 | OPPO/vivo | 三星 | 谷歌 Pixel |
|------|------|------|-----------|------|------------|
| ISO 范围 | 50-409600 | 50-6400 | 100-6400 | 50-3200 | 须通过第三方 |
| 快门范围 | 30s-1/4000 | 30s-1/8000 | 30s-1/4000 | 30s-1/12000 | ❌ |
| 光圈控制 | ✅ (可变光圈) | ❌ | ❌ | ❌ | ❌ |
| RAW | ✅ | ✅ | ✅ | ✅ | ✅ (DNG) |
| 手动对焦 | ✅ | ✅ | ✅ | ✅ | ❌ |
| 测光模式 | ✅ (3种) | ✅ (3种) | ✅ (3种) | ✅ (3种) | ❌ |
| 白平衡K值 | ✅ | ✅ | ✅ | ✅ | ❌ |
| 曝光包围 | ✅ | ❌ | ❌ | ✅ | ❌ |

> Phase 2 实现：APP 启动时检测设备型号 → 查询能力表 → 只推荐该设备支持的参数项。
> 无专业模式的手机（如 Pixel）→ 强制使用小白模式。
> 数据来源：各品牌官网 + 真机实测 + 社区录入。
